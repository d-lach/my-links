# My Links

Urls customization service made for educational purposes with node.js and express.

###### Setup:
1. `cp docker/.env.example docker/.env`
2. fill up `docker/.env`
3. `make install`
4. fill up `server/.env` and `test/.env`

###### Then to run:
`make dev`

###### testing:
`make test`
requires running dev container
