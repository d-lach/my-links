# My Links

Urls customization service made for educational purposes with node.js and express.

###### Setup:
1. `cp .env.example .env`
2. fill up `.env`
3. `make install`
4. fill up `my-links-src/.env` and `my-links-src/.test.env`

###### Then to run:
`make dev`

###### testing:
`make test`
requires runing dev container
